## ğŸ¯ Design Best Practices

### Typography Hierarchy
1. **H1 (Hero)**: 3.5rem - Main message
2. **H2 (Sections)**: 2.5rem - Section headers
3. **H3 (Cards)**: 1.5rem - Card titles
4. **Body**: 1rem - Regular text
5. **Small**: 0.85rem - Labels, metadata

### Spacing System
- **XXS**: 4px
- **XS**: 8px
- **SM**: 16px
- **MD**: 24px
- **LG**: 32px
- **XL**: 48px
- **XXL**: 64px

### Shadows
- **Small**: 0 2px 8px rgba(0,78,124,0.1)
- **Medium**: 0 4px 16px rgba(0,78,124,0.15)
- **Large**: 0 8px 32px rgba(0,78,124,0.2)

### Border Radius
- **Small**: 8px - Inputs, buttons
- **Medium**: 12px - Cards, images
- **Large**: 20px - Sections, modals
- **Pill**: 50px - CTA buttons, badges
